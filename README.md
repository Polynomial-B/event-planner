# Event App

An event searching web application built with **Next.js**.

## Features

- Browse and view events
- Search events by region or 'all'

## Tech Stack

- **Framework:** Next.js
- **Animations:** Framer Motion (used for scroll effects and UI transitions)
- **Styling:** Tailwind CSS + tailwind-merge
- **Icons:** Radix UI Icons
- **Database ORM:** Prisma
- **Validation:** Zod
- **Utilities:** clsx
